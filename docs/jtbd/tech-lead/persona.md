# Tech Lead / Consultant

## Who

Responsible for code quality and governance across teams or client engagements. Installs plugins for their team or recommends them to clients. Represents Windy Road Technology's brand when recommending the suite.

## Context Constraints

- Needs consistent standards across multiple projects and team members
- May install the full suite and configure policy files
- Uses retrospectives, problem management, and incident management for continuous improvement
- Runs pre-flight governance checks before releases, client handovers, and retrospectives — outside normal hook gate triggers

## Pain Points

- Agents bypass reviews with no way to enforce automatically
- No auditability of AI-assisted work
- Inconsistent standards when multiple people use AI agents
- No way to run a governance pre-flight before a release or client handover — assessments only trigger at gate points, by which point the decision to release is already made
