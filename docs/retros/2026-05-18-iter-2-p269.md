Cross-turn staging interference is a class P-fix.
- **P141 changeset-discipline initially blocked the rollout commit**. The convention I cited covers READMEs only; plugin.json modifications are per-package source. Routing target: docs amendment to ADR-021 or `.changeset/` convention to explicitly call out the plugin.json-vs-README boundary.
